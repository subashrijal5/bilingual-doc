# API とエクスポート
プロジェクトの文字列にアクセスするには、API トークンを生成する必要があります。文字列を生成するには、ラベルと有効期限を入力できます。

生成された API キーをコピーできます。


import Callout from "../components/Callout"

<Callout>
    API トークンはデータを安全に保つために安全に保管してください。
</Callout>
