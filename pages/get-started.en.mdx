# Get Started

Let's get started.
At first Please visit [LangLift Pro](https://langlift.com) and [Create an Account](https://langlift.com/signup).

After successful login, Navigate to Projects from the sidebar.

<Image
    src="/project-sidebar.png"
    alt="Project Sidebar"
    width={400}
    height={300}
/>{" "}

### Create New Project

Create New Project selecting the base language that you want to translate from.

<Image
    src="/create-project.png"
    alt="Project Sidebar"
    width={400}
    height={300}
/>{" "}

Then you will be redirected to the following screen where you can add the languages by Clicking **Add Language** Button.

<Image
    src="/project-languages.png"
    alt="Translate Language"
    width={500}
    height={500}
/>{" "}

Or you can start to add the strings that you want to translate by clicking on the **Base language** button that you have selected.

### Add strings

After clicking the Base Language You will be redirected to following page. Where you can add your strings which you want to translate.

<Image
    src="/add-string.png"
    alt="Translate Language"
    width={500}
    height={500}
/>

Here you can create multiple strings and save it to translate. After Saving all of the Strings will be translated on the background automatically. After Successfull completion of translation, you will be notified via email and on the Dashboard notification.
